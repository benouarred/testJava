Montrez nous comment vous développez une application Web moderne !

Votre objectif

  • Développer la partie Front End d'une application TodoList
  • Un serveur node.js expose des ressources REST
    • GET /todos
    • GET /todos/:id
    • POST /todos
    • PUT /todos/:id
    • DELETE /todos/:id
  • Votre application doit pouvoir
    • Afficher la liste des todos
    • Afficher les détails d'un todo
    • Créer un todo
    • Supprimer un todo
    • Mettre à jour un todo
  • Vous avez libre choix des frameworks et librairies
  • Les fichiers statiques sont dans le répertoires public

Critères d'évaluations

Une attention particulière sera apportée à :
  • Les fonctionnalités implémentées
  • Qualité du code
  • Maintenabilité
  • Evolutivité
  • Applications des bonnes pratiques de développements Front End
  • Design de l'application
